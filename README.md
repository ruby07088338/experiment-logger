# experiment-logger
## 📌 開発の背景と目的

【背景：実験中のアナログ作業によるタイムロス】
中和滴定などの化学実験では、滴定曲線の全体像（外形）を把握し、pHが急激に変化するポイントを予測しながら進める必要があります。
これまでは、実験を行いながらノートに数値をメモし、手書きで簡易的なグラフを作成してトレンドを掴んでいました。しかし、この手作業には時間と手間がかかり、本来集中すべき「実験操作」のタイムロスに繋がるという非効率さがありました。

【目的：リアルタイムな可視化による現場の効率化】
そこで、実験中でも数値を入力するだけで、即座にグラフ（滴定曲線）が自動描画されるWebアプリケーションを開発しました。
手作業によるグラフ化の手間を完全に省き、瞬時にデータの外形を可視化することで、実験の精度向上と大幅な作業効率化を実現することを目的としています。

## 💡 解決した課題（本アプリ導入のメリット）

1. 手作業の排除（タイムロスの削減）
   実験中に数値を方眼紙にプロットし、手書きでグラフ化する手間を完全にゼロにしました。これにより、実験中の貴重な時間を「本来の実験操作」や「結果の考察」に集中させることができます。

2. リアルタイムなトレンド把握（意思決定の迅速化）
   数値を入力した瞬間にグラフが自動更新されるため、「どこでpHが急激に変化するか（当量点付近）」を視覚的かつ即座に予測できるようになりました。これにより、滴下スピードを慎重にするタイミングなどが直感的に判断できます。

3. シームレスなデータ移行（二度手間の解消）
   入力した滴下量とpHのデータは、裏側で自動的にCSVファイルとして蓄積されます。紙のメモから後でExcelに打ち直すというアナログな二度手間がなくなり、実験後のレポート作成もスムーズに行えるようになりました。

## 🛠 使用技術
- **Language**: Python 3.14.3
- **Framework**: Flask
- **Frontend**: HTML5, CSS3, Chart.js (グラフ描画ライブラリ)
- **Database**: CSV (データのポータビリティを重視)

## 📊 実際の活用事例
実際に大学の実験（リン酸の滴定）で使用したデータを蓄積しています。
